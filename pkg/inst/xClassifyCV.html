xClassifyCVR Documentation

Function to visualise cross-validation performance

Description

xClassifyCV is supposed to visualise cross-validation performance. It returns an object of class "ggplot".

Usage

xClassifyCV(list_sClass, displayBy = c("ROC", "Fmax"), type = c("bar",
"ridge"), sort = TRUE, colormap = "spectral", ncolors = 64, ...)

Arguments

list_sClass

a list of "sClass" objects or an object of class "sClass"

displayBy

which statistics will be used for displaying. It can be "ROC" for AUC in ROC (shaped by direction), "Fmax" for F-max in Precision-Recall curve

type

the plot type. It can be "bar" and "ridge"

sort

logical to indicate whether to sort methods according to performance. By default, it sets TRUE

colormap

short name for the colormap. It can be one of "jet" (jet colormap), "bwr" (blue-white-red colormap), "gbr" (green-black-red colormap), "wyr" (white-yellow-red colormap), "br" (black-red colormap), "yr" (yellow-red colormap), "wb" (white-black colormap), and "rainbow" (rainbow colormap, that is, red-yellow-green-cyan-blue-magenta). Alternatively, any hyphen-separated HTML color names, e.g. "blue-black-yellow", "royalblue-white-sandybrown", "darkgreen-white-darkviolet". A list of standard color names can be found in http://html-color-codes.info/color-names

ncolors

the number of colors specified over the colormap

...

additional graphic parameters for ggridges::geom_density_ridges_gradient (such as scale=1)

Value

an object of class "ggplot"

Note

none
