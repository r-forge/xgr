xClassifyParaR Documentation

Function to visualise tuning parameters

Description

xClassifyPara is supposed to visualise tuning parameters.

Usage

xClassifyPara(sClass, colormap = "spectral", ncolors = 50,
contour = TRUE, plot.3D = FALSE, theta.3D = 40, phi.3D = 25,
text.3D = TRUE)

Arguments

sClass

an object of class "sClass". Alternatively, it can be a data frame

colormap

short name for the colormap. It can be one of "jet" (jet colormap), "bwr" (blue-white-red colormap), "gbr" (green-black-red colormap), "wyr" (white-yellow-red colormap), "br" (black-red colormap), "yr" (yellow-red colormap), "wb" (white-black colormap), and "rainbow" (rainbow colormap, that is, red-yellow-green-cyan-blue-magenta). Alternatively, any hyphen-separated HTML color names, e.g. "blue-black-yellow", "royalblue-white-sandybrown", "darkgreen-white-darkviolet". A list of standard color names can be found in http://html-color-codes.info/color-names

ncolors

the number of colors specified over the colormap

contour

logical to indicate whether 2D coutour plot should be also included

plot.3D

logical to indicate whether to plot in 3D

theta.3D

the azimuthal direction. By default, it is 40

phi.3D

the colatitude direction. By default, it is 20

text.3D

logical whether to text the point with the maximum value

Value

a ggplot object (1D), a trellis object (2D), and NULl (3D)

Note

none
