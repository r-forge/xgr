xBicreateR Documentation

Function to create a bipartitle graph

Description

xBicreate is supposed to create a bipartitle graph.

Usage

xBicreate(data, verbose = TRUE)

Arguments

data

a data frame/matrix to create a bipartitle graph. The coloring can be continuous (numeric matrix) or discrete (factor matrix)

verbose

logical to indicate whether the messages will be displayed in the screen. By default, it sets to true for display

Value

It returns an igraph object.

Note

none

See Also

xBicreate

Examples

data <- cbind(c(1,1,0,1), c(0,1,2,0))
ig <- xBicreate(data)
