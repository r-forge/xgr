xWGCNAtomR Documentation

Function to draw TOM plot

Description

xWGCNAtom is supposed to draw TOM plot from an object of class "cModule".

Usage

xWGCNAtom(cModule, beta_pseudo = NULL, which.modules = NULL,
colormap = "grey-orange-red", zlim = c(0, 1),
filename = "xWGCNAtom", verbose = T)

Arguments

cModule

an object of class "cModule"

beta_pseudo

NULL or an integer used to enhance TOM plot. If NULL, beta will be the one determined internally. It can be specified explicitly by the beta value (an integer)

which.modules

an integer vector specifying which modules are visualised. If NULL (by default), all modules will be used

colormap

short name for the colormap. It can be one of "jet" (jet colormap), "bwr" (blue-white-red colormap), "gbr" (green-black-red colormap), "wyr" (white-yellow-red colormap), "br" (black-red colormap), "yr" (yellow-red colormap), "wb" (white-black colormap), and "rainbow" (rainbow colormap, that is, red-yellow-green-cyan-blue-magenta). Alternatively, any hyphen-separated HTML color names, e.g. "blue-black-yellow", "royalblue-white-sandybrown", "darkgreen-white-darkviolet". A list of standard color names can be found in http://html-color-codes.info/color-names

zlim

the minimum and maximum z values for which colors should be plotted, defaulting to [0,1]

filename

the without-extension part of the name of the output file. By default, it is 'xWGCNAtom'

verbose

logical to indicate whether the messages will be displayed in the screen. By default, it sets to true for display

Value

a named vector specifying module colors

Note

none
