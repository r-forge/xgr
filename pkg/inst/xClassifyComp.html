xClassifyCompR Documentation

Function to compare prediction performance results

Description

xClassifyComp is supposed to compare prediction performance results. It returns an object of class "ggplot".

Usage

xClassifyComp(list_pPerf, displayBy = c("ROC", "PR"), type = c("bar",
"curve"), sort = FALSE, detail = TRUE, facet = FALSE)

Arguments

list_pPerf

a list of "pPerf" objects or a "pPerf" object

displayBy

which performance will be used for comparison. It can be "ROC" for ROC curve (by default), "PR" for PR curve

type

the type of plot to draw. It can be "bar" for bar plot (by default) or "curve" for curve plot

sort

logical to indicate whether to sort methods according to performance. By default, it sets FALSE

detail

logical to indicate whether to label performance and score direction (together with methods). By default, it sets TRUE. Only works for the curve

facet

logical to indicate whether to facet/wrap a 1d of panels into 2d. By default, it sets FALSE. Only works for the curve

Value

an object of class "ggplot" or NULL (if all input pPerf objects are NULL)

Note

none

xClassifyPerf
