xCEMiToolR Documentation

Function to post-process results from CEMiTool

Description

xCEMiTool is supposed to post-process results from CEMiTool (an object of class "CEMiTool").

Usage

xCEMiTool(cem, filename = "xCEMiTool", verbose = T)

Arguments

cem

an object of class "CEMiTool"

filename

the without-extension part of the name of the output file. By default, it is 'xCEMiTool'

verbose

logical to indicate whether the messages will be displayed in the screen. By default, it sets to true for display

Value

a data frame with three columns 'genes' (ordered by intramodular connectivity), 'modules' (M1, M2, ...) and 'rank' (integer with a module).

Note

none

See Also

xCEMiTool

Examples

## Not run: 
# Load the library
library(XGR)
RData.location <- "http://galahad.well.ox.ac.uk/bigdata/"

cem <- cemitool(expr, filter=FALSE, merge_similar=TRUE, min_ngen=70,
diss_thresh=0.8, verbose=T)
df_res <- xCEMiTool(cem)
